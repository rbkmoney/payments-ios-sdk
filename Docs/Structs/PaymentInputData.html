  • payerEmail

    Email плательщика.

  • allowedPaymentMethods

    Разрешенные методы оплаты.

    Методы оплаты, которые разрешено использовать для оплаты инвойса. Конечный список методов оплаты, отображаемый пользователю в UI, будет зависеть от перечисленных в этом параметре методов и от требований, накладываемых каждым методом оплаты в отдельности. Требования указаны в описании перечисления PaymentMethod.

    Например, можно разрешить оплату только банковской картой, при этом другие методы не будут отображены в UI даже при возможности их применения.
